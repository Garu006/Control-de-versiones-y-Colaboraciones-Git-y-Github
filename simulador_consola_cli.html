1. Reconocimiento

Descubre dónde estás y qué hay a tu alrededor.

$ pwd
$ ls
$ date

2. Estructura

Prepara el terreno para el proyecto.

$ mkdir web-app
$ cd web-app
$ touch index.html

3. Edición

Crea contenido real en los archivos.

$ echo "Hola Git" > readme.txt
$ cat readme.txt

4. Organización

Mueve y respalda archivos.

$ cp readme.txt backup.txt
$ mkdir docs
$ mv readme.txt docs/

5. Limpieza

Elimina archivos y carpetas.

$ rm backup.txt
$ cd ..
$ rm -r web-app
Tip: Usa history para ver tus comandos anteriores.
